The official score-sheet .


The score-sheet is made up of two sheets that are identical except forthe column headings.

The first sheet contains all the information on the game and a table fornoting down runs per inning. This first sheet is reserved for the visitingteam, or whichever team bats first.

The heading of the second sheet is different from that of the first.

The left-hand side contains a repetition of the game data, to identifywhich game the second sheet refers to.

There is a space to put the number of spectators (attendance), the weather type and a space for notes, which should be used for recording the circumstances of interruptions to the game, reasons for any protests or reservations, and an explanation of any plays that could be difficult to interpret.
